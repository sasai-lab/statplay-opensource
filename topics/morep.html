二項分布・ポアソン分布・指数分布

成功回数の二項分布、稀な事象の発生回数のポアソン分布、無記憶の待ち時間を表す指数分布。離散と連続の橋渡しをスライダーで体感する。

P.04 / DISCRETE + EXPONENTIAL

離散分布と指数分布 — 数える世界の確率モデル

正規分布は連続で対称。でも現実のデータはそうとは限らない。コイン投げは離散、来客数はポアソン、待ち時間は指数分布。残りの分布で道具箱を完成させよう。
統計の入り口でよく登場する二項分布・ポアソン分布・指数分布の三つ。成功回数・事象発生回数・待ち時間——離散と連続の橋渡しをスライダーで体感する。
実験ガイド — 二項分布
  1. Step 1: n=20, p=0.5 → 左右対称のベル型。p を 0.1 に → 右に歪む。
  2. Step 2: n を 50、p を 0.06 に → np≈3。下のポアソン(λ=3)とほぼ同じ形になる。
  3. Step 3: n を大きく、p を小さく保つ → ポアソンに収束していく過程が見える。
二項分布 B(n, p)

// ここで使われる公式

具体例で読む（10回コインを投げて表が3回出る確率）
・p=0.5, n=10, k=3 のとき——
・p³ = 0.5³：「3回とも表が出る」確率
・(1−p)⁷ = 0.5⁷：「残り7回ぜんぶ裏」の確率
・この2つを掛けると「表表表裏裏裏裏裏裏裏」という特定の並びの確率になる
・でも表が3回出る並びは他にもある（表裏表裏裏…とか）。全部で ₁₀C₃ = 120通り
・だから P(X=3) = 120 × 0.5³ × 0.5⁷ ≈ 0.117

覚えておくと得する性質
・平均 = np（10回×0.5 = 5回、直感どおり）
・分散 = np(1−p)。p=0.5 のとき分散が最大 ← 結果が一番読めないのは五分五分のとき

実験ガイド — ポアソン分布
  1. Step 1: λ=1 → 0付近に集中した急降下の形。λ=5 → ベル型に近づく。
  2. Step 2: λ を 20 まで上げる → 正規分布のベル型にそっくり（CLTの予告）。
  3. Step 3: λ が平均でもあり分散でもある。λ=3 にしてグラフ左上の「E[X] = Var[X] = λ = 3.00」で確認。

λ が大きくなるにつれ、ポアソンは正規分布に近づく。

// ここで使われる公式

具体例で読む（1時間に平均3件メールが来る。5件来る確率は？）
・λ=3, k=5 のとき——
・λ⁵ = 3⁵ = 243：「5件分のパワー」（λが大きいほど多く来やすい）
・e⁻³ ≈ 0.050：「0件も来ない確率」が基準。λが大きいほどここが小さい
・5! = 120：5件の来る順番を区別しない ← メールAが先かBが先かは気にしない
・P(X=5) = 243 × 0.050 / 120 ≈ 0.101

二項分布との関係 ← 試験で出る
・二項分布で n をめちゃくちゃ大きく、p をめちゃくちゃ小さくして np=λ を保つと → ポアソンに化ける
・上のシミュレーションで n=50, p=0.06 にしてみよう。λ=3 のポアソンとほぼ同じ形になる
・最大の特徴：平均 = 分散 = λ。これが成り立たないデータにポアソンを当てはめると失敗する

実験ガイド — 指数分布
  1. Step 1: λ=1 → 平均待ち時間1。λ=2 → 平均0.5。λ が大きいほど「すぐ起きる」。
  2. Step 2: λ を 0.1 に → 裾が長い。稀にしか起きない事象の待ち時間。
  3. Step 3: 無記憶性：もう10分待っても、あと何分待つかは今来た人と同じ。
指数分布 Exp(λ) — 待ち時間
いつ使う？ → 次の電話が来るまでの待ち時間、電球が切れるまでの寿命
1
無記憶性：バスを10分待っても、あと何分待つかは今来た人と同じ。過去の待ち時間は将来に影響しない。

// ここで使われる公式

具体例で読む（平均5分に1本バスが来る。10分以上待つ確率は？）
・λ=1/5（1分あたり0.2本）、t=10 のとき——
・P(X > 10) = e⁻⁰·²×¹⁰ = e⁻² ≈ 0.135 → 約13.5%
・右の式は「t分以上待っちゃう確率」。t が大きいほど急激にゼロに近づく

ポアソン分布との裏表の関係
・ポアソンは「一定時間に何回来るか」（回数の分布）
・指数分布は「次に来るまで何分待つか」（時間の分布）
・同じλを"回数"で見るか"待ち時間"で見るかの違いだけ

無記憶性 ← これが指数分布の最大の特徴
・すでに10分待ったとしても、「あと5分以内に来る確率」は最初から数えた場合と同じ
・「もう長く待ったからそろそろ来るはず」は通用しない
・現実にこれが成り立つのは「発生率が時刻によらず一定」な場合だけ（例：放射性崩壊）
・バスや電車は時刻表があるので、実際には無記憶ではない

// 3つの分布のつながり — バラバラに覚えたら損

この3つ、参考書だと別のページに書いてあるから「3種類の独立した分布」に見えるけど、実はぜんぶつながっている。自分もこれに気づいたとき、急に頭の中が整理された。

二項分布 ──n大・p小──→ ポアソン分布 │ │ │ np, n(1-p)ともに大 │ 「次までの ↓ ↓ 待ち時間」 正規分布で近似 指数分布
  • 二項 → ポアソン：不良品率0.5%の工場で200個検査（n=200, p=0.005）→ λ=1 のポアソンで近似
  • ポアソン ↔ 指数：「1時間にλ回来る」のがポアソン、「次が来るまでの時間」が指数。同じ現象の表と裏
  • 二項 → 正規：np ≥ 5 かつ n(1−p) ≥ 5 なら正規分布 N(np, np(1−p)) で近似できる（CLTの帰結）

// よくある誤解

❌「ポアソン分布はいつでも二項分布の代わりに使える」

p が小さく n が大きいときだけ。p=0.5, n=20 のコイン投げにポアソン近似したら全然合わない。上のシミュレーショ���で実際に比べてみよう。

❌「もう10分待ったんだから、そろそろバスが来るはず」

指数分布（無記憶性）を前提にすると間違い。ただし現実のバスには時刻表があるので、現実のほうが「そろそろ来る」は正しいことが多い。指数分布が当てはまるのは「完全にランダムに起きる事象」だけ。

❌「二項分布の分散は np」

np は平均。分散は np(1−p)。(1−p) の掛け忘れは試験の定番トラップ。p が 0 や 1 に近いほど分散は小さくなる（結果がほぼ確定するから当然）。

// 試験での出方

  • 余事象パターン：「3回以上成功する確率は？」→ P(X≥3) = 1 − P(0) − P(1) − P(2) のほうが速い
  • ポアソン近似：「不良品率2%, 100個中3個以上不良品の確率」→ λ=2 でポアソン計算させる
  • 指数分布の計算：「平均寿命1000時間の電球。500時間以上もつ確率は？」→ e⁻⁰·⁵ ≈ 0.607
  • 平均と分散のセット：「次のうち正しいものを選べ」→ ポアソンは平均=分散=λ、二項の分散は np(1−p)
