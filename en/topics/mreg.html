Multiple Regression — Predict with multiple variables

The "line" of simple regression becomes a plane in 3D space when you add variables. Drag to rotate — see how data points align along the surface.

Simple regression used one variable and fit a line. Multiple regression combines two or more variables — the fit becomes a plane. Rotate the 3D plot and see how the plane captures the data.

Experiment Guide — try these in order
  1. Step 1: Drag the 3D plot to rotate it. The translucent surface is the regression plane — data points align along it.
  2. Step 2: Dashed lines from points to the plane are residuals (prediction errors). Shorter = better fit.
  3. Step 3: Move the study slider → the large dot moves along the plane. That's the predicted score.
  4. Step 4: Move sleep too → it moves in a different direction on the plane. Two variables combine for prediction.

Each part
・β₀ (intercept): baseline prediction when all variables = 0
・β₁x₁: "holding x₂ fixed, how much does y change per unit of x₁" × the value of x₁
・β₂x₂: same idea — the isolated contribution of x₂

How it differs from simple regression
・Simple regression slope ignores other variables (crude effect)
・Multiple regression β isolates each variable's "pure effect" with others held constant (partial regression coefficient)
・Even if study hours and sleep correlate, multiple regression separates their independent contributions

Geometric picture
・With 2 predictors, the fit is a plane in 3D. Least squares picks the tilt that minimizes total squared distance from points to plane
