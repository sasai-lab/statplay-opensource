// ここで使われる公式

左の式：条件付き確率 P(A|B)
・P(A∩B)：A と B が同時に起きる確率（ベン図の重なり部分）
・÷ P(B)：「B が起きた」という条件で世界を絞り込む
・意味：B が起きたと分かったあと、その中で A も起きている割合

右の式：加法定理 P(A∪B)
・P(A) + P(B)：単純に足すと、重なり（A∩B）を二重カウントしてしまう
・− P(A∩B)：二重カウント分を引いて正しい値にする
・排反（A∩B = 0）なら単純に足すだけでOK

独立との関係
・A と B が独立 ⇔ P(A|B) = P(A)（B を知っても A の確率が変わらない）
・このとき P(A∩B) = P(A)·P(B) が成り立つ（掛け算で求まる）

// Formula used here

Left: Conditional probability P(A|B)
・P(A∩B): probability both A and B happen (the overlap in a Venn diagram)
・÷ P(B): restrict the universe to "B happened"
・Meaning: within the world where B occurred, what fraction also had A?

Right: Addition rule P(A∪B)
・P(A) + P(B): naively adding double-counts the overlap
・− P(A∩B): subtract it once to correct
・If mutually exclusive (A∩B = 0): just add — no overlap to worry about

Connection to independence
・A and B independent ⇔ P(A|B) = P(A) (knowing B doesn't change A's probability)
・Equivalent: P(A∩B) = P(A)·P(B) — the multiplication rule

// よくある誤解

❌「排反 = 独立」← 最頻出の混同

排反：A∩B = ∅（サイコロで1の目と2の目は同時に出ない）
独立：P(A∩B) = P(A)P(B)（1回目が1の目でも2回目の確率は変わらない）
排反な2事象（P>0）は独立ではない。AとBが同時に起きないなら、Aが起きた時点でBの確率は0に下がるから。

❌「条件付き確率の分母は P(A∪B)」

P(A|B) の分母は P(B)。「Bが起きた世界」に絞り込むから、Bの確率で割る。∪ ではない。

❌「P(A∪B) = P(A) + P(B) はいつでも成立」

排反のときだけ。一般には重なり P(A∩B) を引く必要がある。引き忘れると二重カウントで確率が1を超えることもある。

// Common misconceptions

❌ "Mutually exclusive = independent" — the #1 mix-up

Mutually exclusive: A∩B = ∅ (rolling a 1 and a 2 on the same die).
Independent: P(A∩B) = P(A)P(B) (first roll being 1 doesn't affect the second roll).
Two mutually exclusive events (with P > 0) are never independent. If A happens, B's probability drops to 0 — that's the opposite of independence.

❌ "The denominator of conditional probability is P(A∪B)"

P(A|B) divides by P(B), not P(A∪B). You're restricting the universe to "B happened," so you divide by B's probability.

❌ "P(A∪B) = P(A) + P(B) always holds"

Only when A and B are mutually exclusive. In general, you must subtract P(A∩B) to avoid double-counting. Forgetting this can push probabilities above 1.

// 試験での出方

  • ベン図の読み取り：3つの領域の数値から P(A|B), P(A∪B) を計算
  • 独立の判定：「P(A)=0.3, P(B)=0.4, P(A∩B)=0.12 → 独立か？」→ 0.3×0.4=0.12 → 独立
  • 余事象：「少なくとも1回当たる確率」= 1 − (全部はずれる確率)
  • 排反と独立の違いを正誤問題で問われる

// How it shows up in exams

  • Venn diagram reading: given three region values, compute P(A|B) and P(A∪B)
  • Independence check: "P(A)=0.3, P(B)=0.4, P(A∩B)=0.12 — independent?" → 0.3×0.4 = 0.12 → yes
  • Complement trick: "probability of at least one hit" = 1 − P(all misses)
  • Mutually exclusive vs. independent: true/false questions test whether you confuse the two
